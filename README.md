Black Jack Game
===

## Prerequistes

After checkout run
    $ rvm install jruby
    $ rvm use jruby
    $ bundle install

## Usage

    Rake version:
    $> bundle exec rake -T
    $> bundle exec rake db:game              # starts a new black jack game

    Gem version:
    $> gem install bjg
    $> bjg
